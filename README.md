### 功能概述：


#### 商品上架
支持：商品名称、价格、鞋码（非选择，手动输入）、缩略图
#### 商品查找
按照：pid查找、关键字模糊查询、分页、价格区间、分类
#### 商品排序
按照上架时间排序（默认）、按价格


#### 关于商品分类
添加分类，并指定该分类是否在首页导航展示

#### 前端显示
商品列表标题描述，显示一行（超出字数部分直接断掉）
